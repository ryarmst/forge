Workflow
  1. Set your Collaborator Server hostname below.
  2. Create a New Session — this generates a cryptographic BIID that the server will recognize.
  3. Click Generate Payload to derive subdomains linked to your session. Use these as OOB payloads in testing.
  4. Poll to retrieve DNS, HTTP, and SMTP interactions recorded against your payloads.
This client implements the Collaborator protocol natively — no Burp Suite required. You can also Import BIID to resume a session captured from Burp traffic.
